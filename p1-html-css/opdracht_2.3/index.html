Waarom noem je de eerste HTML-pagina; index.html?

Omdat als een website zonder een specifiek bestand wordt gezocht, dan gaat de browser automatisch zoeken naar de standaardpage of homepage.(oftwel index html) als je je start pagina niet zo noemt is er een kans dat de browser niet je homepagina kan vinden daarom is het belangrijk om je bestand altijd index.html te noemen.
